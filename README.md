# CTIP152 – Computational Thinking & Introduction to Programming

Python exercises, classwork, and assignments completed as part of the
CTIP152 module at STADIO Higher Education.

## About
This repository documents my hands-on Python work from CTIP152, progressing
from basic programming fundamentals to building fully functional CLI applications.

## Projects & Assignments

### 📚 Digital Archive Portal (`main.ipynb`)
A library management system with librarian registration, secure login with a
3-attempt lockout, and a full inventory dashboard. Features staff ID and
password validation, book cataloguing, search, deletion, and auto-generated
accession numbers.

### 🎓 Student Marks Manager (`p2.py`)
A student records system that supports adding students and marks, sorting
records alphabetically, removing entries, and generating class statistics
including average, highest/lowest mark, and a pass/fail breakdown.

### 💰 Budget Planner (`Budget_planner.py`)
A personal finance tracker with categorised expense entry across Food,
Transport, Personal, Entertainment, and Other. Includes custom spending
thresholds with overspend warnings, balance calculation, category search,
and CSV report generation.

## Skills Demonstrated
- Functions, loops, and decision structures
- Dictionary and list manipulation
- Input validation and error handling (`try/except`)
- File I/O (CSV generation)
- Menu-driven CLI application design
